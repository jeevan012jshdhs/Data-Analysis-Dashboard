# Blinkit-Store-Data-Analysis(Interactive Dashboard creation using PowerBI)

## Project Objective
The primary objective of this project is to conduct a comprehensive analysis of Blinkit's sales performance, customer satisfaction, and inventory distribution to identify key growth drivers and operational optimization opportunities.
